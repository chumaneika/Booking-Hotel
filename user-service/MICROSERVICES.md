# user-service — микросервис

Сервис является частью микросервисной архитектуры **Booking Hotel**.

## Роль в системе

- **Назначение:** управление пользователями (создание, обновление, поиск, проверка активности).
- **Регистрация:** подключается к Eureka (при `eureka.client.enabled=true`) для обнаружения сервисов.
- **Порт по умолчанию:** 8081.

## Интеграция с другими сервисами

Другие микросервисы (booking-service, hotel-service и т.д.) могут вызывать user-service по имени `user-service` через Eureka или по прямому URL для проверки пользователя перед бронированием.

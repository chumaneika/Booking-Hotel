# Booking Hotel System

Микросервисная система для бронирования отелей.

Проект состоит из нескольких сервисов, отвечающих за управление пользователями,
каталогом отелей и бронированием.

## Architecture

Проект построен по микросервисной архитектуре.

Сервисы:

- **user-service** — управление пользователями и аутентификация (JWT)
- **catalog-service** — управление каталогом отелей
- **booking-service** — управление бронированиями
- **api-gateway-service** — API Gateway
- **notification-service** — уведомление и оповещение пользователей
- **payment-service** — сервис обработки платежей
- **review-service** — отзывы отелей
- **admin-service** — администрирование
- **service-discovery** — eureka server, библиотека сервисов
